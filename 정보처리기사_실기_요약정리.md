## 4. 데이터베이스 (SQL)

### 4.1 기본 조회 (SELECT)

```sql
-- 기본 조회
SELECT * FROM 테이블명;
SELECT 컬럼1, 컬럼2 FROM 테이블명;

-- 조건 (WHERE)
SELECT * FROM 테이블명 WHERE 조건;
SELECT * FROM 학생 WHERE 나이 > 20;

-- 정렬 (ORDER BY)
SELECT * FROM 테이블명 ORDER BY 컬럼 ASC;  -- 오름차순
SELECT * FROM 테이블명 ORDER BY 컬럼 DESC; -- 내림차순

-- LIMIT
SELECT * FROM 테이블명 LIMIT 10; -- 상위 10개
```

### 4.2 집계 함수

```sql
SELECT COUNT(*) FROM 테이블명;      -- 행 개수
SELECT SUM(컬럼) FROM 테이블명;      -- 합계
SELECT AVG(컬럼) FROM 테이블명;      -- 평균
SELECT MAX(컬럼) FROM 테이블명;      -- 최대값
SELECT MIN(컬럼) FROM 테이블명;      -- 최소값

-- GROUP BY
SELECT 부서, AVG(급여) 
FROM 직원 
GROUP BY 부서;

-- HAVING (GROUP BY 결과 필터링)
SELECT 부서, AVG(급여) 
FROM 직원 
GROUP BY 부서 
HAVING AVG(급여) > 3000;
```

### 4.4 서브쿼리

```sql
-- 단일 행 서브쿼리
SELECT * 
FROM 테이블명

-- 다중 행 서브쿼리 (IN)
SELECT * 
FROM 테이블명 
WHERE 컬럼 IN (SELECT 컬럼 FROM 테이블2 WHERE 조건);

-- EXISTS
SELECT * 
FROM 테이블1 t1
WHERE EXISTS (SELECT 1 FROM 테이블2 t2 WHERE t2.키 = t1.키);
```

### 4.5 데이터 조작 (DML)

```sql
-- INSERT
INSERT INTO 테이블명 (컬럼1, 컬럼2) VALUES (값1, 값2);
INSERT INTO 테이블명 VALUES (값1, 값2, 값3);

-- UPDATE
UPDATE 테이블명 
SET 컬럼1 = 값1, 컬럼2 = 값2 
WHERE 조건;

-- DELETE
DELETE FROM 테이블명 WHERE 조건;
```

### 4.6 데이터 정의 (DDL)

```sql
-- CREATE TABLE
CREATE TABLE 테이블명 (
    컬럼1 데이터타입 PRIMARY KEY,
    컬럼2 데이터타입 NOT NULL,
    컬럼3 데이터타입 DEFAULT 값,
    FOREIGN KEY (컬럼2) REFERENCES 다른테이블(컬럼)
);

-- ALTER TABLE
ALTER TABLE 테이블명 ADD 컬럼명 데이터타입;
ALTER TABLE 테이블명 MODIFY 컬럼명 새데이터타입;
ALTER TABLE 테이블명 DROP COLUMN 컬럼명;

-- DROP TABLE
DROP TABLE 테이블명;
```

```java
// 배열 정렬

// 내림차순
Integer[] arr2 = {5, 2, 8, 1, 9};
Arrays.sort(arr2, Collections.reverseOrder());

// 버블 정렬
for (int i = 0; i < arr.length - 1; i++) {
    for (int j = 0; j < arr.length - 1 - i; j++) {
        if (arr[j] > arr[j + 1]) {
            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp;
        }
    }
}

// 선택 정렬
for (int i = 0; i < arr.length - 1; i++) {
    int minIdx = i;
    for (int j = i + 1; j < arr.length; j++) {
        if (arr[j] < arr[minIdx]) {
            minIdx = j;
        }
    }
    int temp = arr[i];
    arr[i] = arr[minIdx];
    arr[minIdx] = temp;
}
```

### 5.2 탐색

```java
// 선형 탐색
public int linearSearch(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) {
            return i;
        }
    }
    return -1;
}

// 이진 탐색 (정렬된 배열)
public int binarySearch(int[] arr, int target) {
    int left = 0;
    int right = arr.length - 1;
    
    while (left <= right) {
        int mid = (left + right) / 2;
        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return -1;
}
```

```java
public int factorial(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);

public int fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
```

```java
// 피보나치 (DP)
public int fibonacciDP(int n) {
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    
    return dp[n];
}
```

## 6. 네트워크

### 6.1 OSI 7계층

1. 물리계층 (Physical): 비트 전송
2. 데이터링크계층 (Data Link): 프레임 전송, 오류 검출
3. 네트워크계층 (Network): 패킷 라우팅 (IP)
4. 전송계층 (Transport): 세그먼트 전송 (TCP/UDP)
5. 세션계층 (Session): 세션 관리
6. 표현계층 (Presentation): 데이터 변환, 암호화
7. 응용계층 (Application): 사용자 인터페이스 (HTTP, FTP)

### 6.2 TCP/IP

- TCP: 연결 지향, 신뢰성 보장, 속도 느림
- UDP: 비연결, 신뢰성 낮음, 속도 빠름

### 6.3 주요 프로토콜

- HTTP/HTTPS: 웹 통신
- FTP: 파일 전송
- SMTP: 이메일 전송
- DNS: 도메인 이름 해석
